# Random Walk Simulator
# Симулятор "случайной прогулки"

Веб-приложение, которое наглядно и доступно демонстрирует, что такое вероятность, риск и почему казино всегда выигрывает. 

## Key concepts catalog/Ключевые понятия: 
1. Bet size / Размер ставки
2. Initial bankroll / Стартовый капитал
3. Win probability / Шанс на выигрыш
4. Number of steps / Количество шагов
5. Realistic Gameplay / Реалистичность игры
6. Parallel Simulations / Количество параллельных симуляций 

**Bet size/Размер ставки:** количество денежного ресурса, расходуемое в случае неудачного шага и получаемое в случае удачного шага

**Initial bankroll/Стартовый капитал:** количество денежного ресурса вначале игры, до момента совершения первой ставки

**Win probability/Шанс на выигрыш:** шанс на то, что каждый отдельный шаг будет успешным. Чем ниже данное значение, тем быстрее наступит потенциальное "разорение". 
Выставление данного параметра выше 50-ти процентов не имеет практического смысла ввиду нереалистичности сценария.

**Number of steps/Количество шагов:** Данное значение стоит трактовать как максимальное кол-во шагов, после которого игра остановится, если "разорение" не наступит раньше.

**Realistic Gameplay/Реалистичность игры:** Проставление данной галочки ограничит максимальный шанс на победу, обозначив его как 50%. Полезно для симуляции реализма. 
В реальных ситуациях игрокам никогда не даются шансы, превышающие 50%, а зачастую они находятся на отметке ниже 50%.

**Parallel Simulations/Количество параллельных симуляций:** Полезно для более наглядной статистики. 
Особенное внимание стоит уделить тому, что для всех параллельных симуляций действуют изначально заданные настройки, но результаты будут иметь тенденцию отличаться друг от друга.
